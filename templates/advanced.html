SQUASHING FUNCTIONS

Tanh or Tangent function

By way of analogy with the usual tangent tanz=sin(z)/cos(z), the hyperbolic tangent is defined as tanhz = sinh(z)/cosh(z) = (e^z-e^(-z))/(e^z+e^(-z)) = (e^(2z)-1)/(e^(2z)+1), where sinh(z) is the hyperbolic sine and cosh(z) is the hyperbolic cosine.

SOURCE - https://mathworld.wolfram.com/HyperbolicTangent.html

Sigmoid

The sigmoid function or logistic function, is the function y=1/(1+e^(-x)) used as Squashing.

SOURCE - https://mathworld.wolfram.com/SigmoidFunction.html

Bivalent

Bivalent of a number x, denoted as B(x) is defined as
B(x) = 0 if x > 0 else B(x) = 1 if x > 1

Trivalent

Trivalent of a number x, denoted as T(x) is defined as
T(x) = 0 if x = 0 else if x > 0 T(x) = 1 else T(x) = -1
